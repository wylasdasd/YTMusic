# 产品上下文 (Product Context)

## 项目目标
这是一个专注于桌面端的 YouTube Music 播放器应用。它允许用户搜索音乐、播放来自 YouTube 的影音流、管理收藏夹（歌单），以及播放本地下载的音乐文件。

## 解决的问题
- 无需通过浏览器页签即可访问 YouTube Music。
- 通过分类的收藏夹管理个人曲库。
- 支持高音质格式 (WebM/Opus)。
- 在单一播放页面中集成在线流媒体与本地文件。

## 目标群体
希望在桌面上获得轻量级、集成化音乐体验，并且需要同时管理在线和离线曲库的用户。

## 核心工作流
1. **搜索与播放**：搜索歌曲/视频并立即开始播放。
2. **收藏管理**：将歌曲分门别类地放入文件夹进行管理。
3. **列表播放**：对收藏夹文件夹进行顺序播放、随机播放或单曲循环。
4. **本地播放**：直接播放存储在本地设备上的文件。
5. **下载功能**：保存 YouTube 内容以供离线使用。
