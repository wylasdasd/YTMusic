# 产品上下文 (Product Context)

> 最后更新：2026-06-29

## 项目目标

YTMusic 是一款跨平台（**Windows + Android** 已适配；iOS / macOS 未验证）YouTube 音乐播放器。用户可搜索、播放 YouTube 影音流、管理收藏夹、下载离线，并通过 AList 与远端存储同步。

## 解决的问题

- 无需浏览器页签即可使用 YouTube 曲库（需自行 VPN）。
- 收藏夹按文件夹组织，支持顺序 / 随机 / 单曲循环。
- 在线播放默认 audio-only（WebM 优先）；可选视频模式。
- Android 支持后台播放与通知栏控制。
- 本地下载与 AList 远端目录下载，统一进入本地库播放。

## 目标群体

希望在桌面或 Android 上获得集成化音乐体验、同时管理在线与离线曲库的用户。

## 核心工作流

1. **搜索与播放**：Home 搜索 → 立即播放 → 全局播放器跨页连续。
2. **收藏管理**：文件夹分类、文件夹内列表播放。
3. **本地播放**：下载页或收藏夹直接播放本地文件。
4. **下载**：音频/视频离线保存；传输页查看任务进度。
5. **AList**：配置远端存储、上传本地文件、从远端目录拉回歌曲。
6. **其他**：历史播放、主题与播放设置（Theme 侧边栏）。

## 信息架构要点

- 底部导航：Favorites / Player / Home / Other。
- Other 聚合：本地资源、传输任务、远程存储、播放历史。
- 播放器分 `/player/audio` 与 `/player/video`；Android 原生全屏视频不进 `/player/video`。
